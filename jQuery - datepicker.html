2011-10-14，Laurens 的 costcontrole 项目用到 datetimepicker，jQuery 官方还没有，但已经有人做出了解决办法。

相关页面：http://trentrichardson.com/examples/timepicker/

js 文件：http://trentrichardson.com/examples/timepicker/js/jquery-ui-timepicker-addon.js

示例步骤：

  1. 依次引入
    < script type="text/javascript" src="resource/jquery-1.6.4.js">
    < script type="text/javascript" src="resource/jquery-ui.js">
    < script type="text/javascript" src="resource/jquery.ui.slider.js">// datetimepicker 用到了 slider 来选择时间
    < script type="text/javascript" src="resource/jquery-ui-timepicker-addon.js">
  2. 因为 jquery-ui-timepicker-addon.js 中有 
    $.extend($.ui, { timepicker: { version: "0.9.7" } });
...
this.hour_slider = $tp.find('#ui_tpicker_hour_'+ dp_id).slider({
    orientation: "horizontal",
    value: this.hour,
    min: o.hourMin,
    max: hourMax,
    step: o.stepHour,
    slide: function(event, ui) {
        tp_inst.hour_slider.slider( "option", "value", ui.value);
        tp_inst._onTimeChange();
    }
});
  3. 依次引入
    < link href="resource/jquery-ui.css" rel="stylesheet" type="text/css" />
    < link href="resource/jquery.ui.datepicker.css" rel="stylesheet" type="text/css" />
  4. 使用： 
    jQuery(function(){
    jQuery('#datetime').datetimepicker();
